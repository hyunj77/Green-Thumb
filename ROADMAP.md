# Green Thumb 로드맵

## 구축 완료
- 인증(회원가입/로그인/로그아웃)
- 마이 그린 도감: 반려식물 등록, 물주기 기록/알림
- 성장일기: 식물별 날짜/키/사진/메모 타임라인 기록
- 커뮤니티 피드: 카테고리별 글 작성/수정/삭제, 댓글, 물주기·햇빛쬐기 반응
- 로컬 장터: 카테고리별 매물 등록/조회
- Home 탭 Contact 카드 + 식물 방명록(초록 엄지 응원 도장)

## 외부 유료 API / 별도 예산·설정이 필요한 기능 (검토 필요)
현재 MyGarden 페이지에 실제 동작하지 않는 **샘플 미리보기 카드**로만 배치되어 있습니다.
실제 구현하려면 아래와 같은 외부 서비스 계약/키 발급/비용 검토가 선행되어야 합니다.

| 기능 | 필요한 것 |
|---|---|
| AI 병해충 진단 | Vision API (예: Google Cloud Vision, OpenAI Vision 등) 키 + 병해충 판별 모델/데이터셋 |
| AI 식물 자동 등록 (사진→품종 인식) | 이미지 분류 API 또는 자체 모델 |
| AI 성장 예측 (CG 이미지 생성) | 이미지 생성 API (예: Stable Diffusion, DALL·E 등) |
| 성장 타임랩스 자동 생성 | 서버 사이드 영상 인코딩 인프라 (클라이언트 전용 구조로는 한계) |
| AI 플랜테리어 추천 | 이미지 인식 + 추천 모델 |
| 우리집 환경 분석 (위치 기반 습도/온도) | 기상청 API 또는 날씨 API 키, 위치 권한 UX |
| 실시간 1:1 채팅 (로컬 장터) | Supabase Realtime 채널 설계 + UX 추가 작업 |
| 정밀 GPS 반경 매칭 | PostGIS 확장 또는 좌표 기반 반경 쿼리 설계 |

우선순위와 예산이 정해지면 하나씩 순서대로 이어서 구현합니다.
